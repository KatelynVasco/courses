# Assignment 2: Basic Bash

## Overview

Four short bash scripting exercises covering quoting, variables, streams, and pipelines. Each exercise is a separate `.sh` file that you'll complete.

## Getting Started

1. Open your GitHub Codespace
2. Navigate to `assignments/Assignment2/`
3. You'll find four starter scripts — each one has instructions in the comments

## The Exercises

### 1. `soda.sh` — Expensive Soda
Print this exact output:
```
A soda costs $9?!
```
The `$9` must appear literally — not as a variable expansion.

### 2. `greetings.sh` — Greetings
Using the variables `first_name` and `last_name` (already set in the starter file), print:
```
Hello
[first_name][TAB][last_name]
How are you?
```
There must be a **tab character** (not spaces) between the first and last name.

### 3. `mixed.sh` — Mixed Messages
Print `Everything is fine` to **stdout** and `WARNING: SEVERE ERROR` to **stderr**.

Both messages should appear when you run the script, but they travel through different streams.

### 4. `pipeline.sh` — Read, Sort and Trim
Given a file called `input.dat` (already created in the starter):
1. Sort the lines in **descending** order
2. Save the sorted result to `sorted.dat`
3. Print to stdout: skip the first line of the sorted result, then output the **next 5 lines**

Think pipelines — this can be done in very few lines.

## Running the Tests

When you're ready to check your work:

```bash
bash run_tests.sh
```

This runs all four exercises, checks the output against expected results, and generates `results.txt` with your code and test results.
